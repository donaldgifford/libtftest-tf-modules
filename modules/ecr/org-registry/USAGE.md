## Inputs

| Name | Description | Type | Default | Required |
| ---- | ----------- | ---- | ------- | :------: |
| helm\_charts\_prefix | ECR repository name prefix for internal Helm charts. The aws\_ecr\_repository\_creation\_template targets this prefix; first `helm push <prefix>/<chart>` auto-creates the repo under the template's policy. | `string` | `"helm-charts"` | no |
| kms\_key\_arn | Optional caller-supplied KMS key ARN for ECR encryption. When null (default), the module creates a key + alias internally. Pattern matches the cluster module's bring-your-own KMS shape. | `string` | `null` | no |
| name\_prefix | Short name prefix for the KMS alias, ECR-template IAM role, and publisher IAM policy. The module composes alias/<name\_prefix>-ecr-oci, <name\_prefix>-ecr-template, and <name\_prefix>-oci-publisher from this value. | `string` | n/a | yes |
| organizations\_org\_id | AWS Organizations ID (o-...) used in the aws:PrincipalOrgID condition on the org-wide pull policy embedded in both creation templates. Caller-supplied — the module does NOT read this via data.aws\_organizations\_organization (per IMPL-0006 Q2 (a) / ADR-0001 explicit-input posture). | `string` | n/a | yes |
| pre\_release\_retention\_days | Days to retain pre-release / dev-tagged images before the ECR lifecycle policy prunes them. Embedded in both creation templates' lifecycle\_policy JSON. | `number` | `90` | no |
| publish\_to\_ssm | When true, the module emits two SSM Parameter Store entries (the publisher policy ARN and the full publisher policy JSON) so consumers can discover the policy programmatically. Default false — opt-in (per IMPL-0006 Q7). | `bool` | `false` | no |
| ssm\_cross\_account\_org\_id | When non-null, the SSM parameters switch to Advanced tier and a resource-based policy grants ssm:GetParameter / ssm:GetParameters to aws:PrincipalOrgID = this value — letting cross-account publisher CI roles read the policy JSON. Null (default) = same-account-only mode (Standard tier, no resource-based policy). | `string` | `null` | no |
| ssm\_parameter\_path\_arn | SSM Parameter Store path for the publisher policy ARN. Same-account consumers read this to discover the policy ARN. Only used when publish\_to\_ssm = true. | `string` | `"/platform/ecr-oci-publisher-policy-arn"` | no |
| ssm\_parameter\_path\_json | SSM Parameter Store path for the publisher policy JSON. Cross-account consumers read the JSON via data.aws\_ssm\_parameter and recreate the policy locally in their own account. Only used when publish\_to\_ssm = true. | `string` | `"/platform/ecr-oci-publisher-policy-json"` | no |
| tags | AWS resource tags applied to every taggable resource in the module (KMS key, IAM role, IAM policy, creation templates, SSM parameters). | `map(string)` | `{}` | no |
| tf\_modules\_prefix | ECR repository name prefix for internal Terraform modules published as OCI artifacts. Same template/auto-creation behavior as helm\_charts\_prefix, distinct prefix. | `string` | `"tf-modules"` | no |
| untagged\_retention\_days | Days to retain untagged images before the ECR lifecycle policy prunes them. Embedded in both creation templates' lifecycle\_policy JSON. | `number` | `7` | no |

## Outputs

| Name | Description |
| ---- | ----------- |
| ecr\_template\_role\_arn | ARN of the ECR-assumed IAM role driving repo creation via the two aws\_ecr\_repository\_creation\_template resources. Provided for observability / cross-stack debugging; consumers should not attach this role themselves. |
| helm\_charts\_template\_id | ID of the aws\_ecr\_repository\_creation\_template for the helm-charts/* prefix. (v6 schema exposes `id`, not `arn`.) |
| kms\_key\_arn | ARN of the KMS key used to encrypt OCI artifact repositories. Module-managed when var.kms\_key\_arn is null; BYO when supplied. |
| publisher\_policy\_arn | ARN of the reusable customer-managed publisher policy. Same-account CI / IRSA roles attach this via aws\_iam\_role\_policy\_attachment. Cross-account consumers recreate the policy locally from the JSON SSM parameter (see publisher\_policy\_ssm\_json\_parameter\_name). |
| publisher\_policy\_ssm\_arn\_parameter\_name | SSM Parameter Store path where the publisher policy ARN was published (or null when var.publish\_to\_ssm = false). Same-account consumers read this via data.aws\_ssm\_parameter to discover the policy ARN dynamically. |
| publisher\_policy\_ssm\_json\_parameter\_name | SSM Parameter Store path where the full publisher policy JSON was published (or null when var.publish\_to\_ssm = false). Cross-account consumers read this via data.aws\_ssm\_parameter and recreate the policy locally in their own accounts (IAM policies don't cross account boundaries by reference). |
| ssm\_org\_read\_policy\_json | Resource-based policy JSON granting org-wide ssm:GetParameter on the two SSM parameters (gated on var.publish\_to\_ssm AND var.ssm\_cross\_account\_org\_id != null; null otherwise). See README. |
| tf\_modules\_template\_id | ID of the aws\_ecr\_repository\_creation\_template for the tf-modules/* prefix. |
